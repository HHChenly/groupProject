上传文件到远程仓库
前提：找到你要提交的代码的根目录，保证网络连接。
1.cd E:/Git_download
2.把远程仓库里面的文件克隆到本地的git_download仓库
  git clone git@github.com:HHChenly/groupProject.git
3.cd groupProject
4.查看本地分支 git branch
5.查看远程和本地所有分支 git branch -a
6.切换到指定分支；命令：git checkout 分支名
7.git add .(或者你要提交的代码文件)
8.git commit -m “描述”
9.git remote add origin 要上传到的目标仓库地址
10.git push -u origin 分支名  推到远程仓库
